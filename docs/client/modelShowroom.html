Function takes a Roles as parameter and returns all Models where the User has the expected Role

userInvitedModels = (role)-> tmpResult = Models.find({}).fetch() modelsResult = [] tmpResult.forEach (i)-> permission = checkModelPermission i._id,false if permission == role modelsResult.push i return modelsResult

Template.modelShowroom.Owner = ()-> userInvitedModels(Roles.owner)
