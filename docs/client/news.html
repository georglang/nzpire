Sort Function

Sorts an array by timestamps
* params: object, object (with .updatedAt)
* return: integer (-1,0,1)

sortArrayByTimestamp = (a,b)-> return -1 if a.updatedAt > b.updatedAt return 1 if a.updatedAt < b.updatedAt return 0
